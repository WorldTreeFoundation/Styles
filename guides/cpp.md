# Style guide for C++

## Folder and files names

    folder_name:
       - file_name_1
       - file_name_2

## Docstrings

```cpp
// Example function's docstring
void example() {}
```

## Variable names

```cpp
int lower_snake_case;
```

## Function names

```cpp
void lower_snake_case() {}
```

## Class names

```cpp
class UpperCamelCase 
{
    void lower_snake_case() {}
    int lower_snake_case_again = 0;
}
```
