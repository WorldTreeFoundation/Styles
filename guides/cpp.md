# Style guide for C++

## Docstrings

```cpp
// Example function's docstring
void example() {}
```

## Variable names

```cpp
int lower_snake_case
```

## Function names

```cpp
void lower_snake_case() {}
```

## Class names

```cpp
class UpperCamelCase 
{
    void lower_snake_case() {}
    int lower_snake_case_again = 0;
}
```
